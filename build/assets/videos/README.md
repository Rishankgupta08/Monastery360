# Video Files for Virtual Tour

Please add your video files to this directory with the following names:

## Required Video Files:

1. **rumtek-outside.mp4** - Outside view video (loops automatically)
   - This will be displayed continuously on the main screen
   - Should show the exterior view of Rumtek Monastery

2. **rumtek-interior.mkv** - Interior view (your MKV file)
   - Activated by clicking the "Interior View" hotspot (MapPin icon)
   - Shows the magnificent interior of the monastery

3. **rumtek-prayer-hall.mp4** - Prayer hall video (your first MP4 file)
   - Activated by clicking the "Prayer Hall" hotspot (Book icon)
   - Shows the sacred prayer hall where monks gather

4. **rumtek-meditation.mp4** - Meditation chamber video (your second MP4 file)
   - Activated by clicking the "Meditation Chamber" hotspot (Users icon)
   - Shows the peaceful meditation spaces

## File Format Support:
- MP4 files (recommended for web compatibility)
- MKV files (supported by modern browsers)

## Notes:
- The outside view video will loop automatically
- Other videos have playback controls when viewing
- Users can return to outside view anytime using the "Back to Outside" button
- Hotspots appear as animated, glowing icons on the outside view
